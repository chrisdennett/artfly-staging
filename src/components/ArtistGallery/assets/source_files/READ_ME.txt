*******************************
Explanation of the SVG workflow
*******************************

- svg_originals
Contains the raw Inkscape files.

- svg_exports
Contains the svg exported from Inkscape using file > save a copy > [save as type: Optimised SVG (*.svg)]

- generated_svgs
Contains the result of the 'svg-convert-AND-optimise' Gulp function.

- component_templates
Contains the files into which Gulp injects the svg markup.

- parent directory
Contains the components generated by the Gulp functions that inject the svg markup and convert the svg attributes to react friendly names.
